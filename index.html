Basics About CSS Styles

    To build a great website, you need to understand how to apply CSS (the "where") and how the browser calculates space (the "box"). Here is a breakdown of the three ways to apply CSS and the fundamental spacing properties.

  1. Three Ways to Apply CSS These methods determine where your style code lives in relation to your HTML. Method Location Best For... Inline Directly inside an HTML tag using the style attribute. Quick fixes or testing a single element. Internal Inside a <style> tag within the <head> of an HTML file. Single-page websites where you want everything in one file. External In a separate .css file linked via the <link> tag. Best Practice. Keeping styles consistent across many pages.
  2. Padding : Padding is the internal space between the content (like text) and the border of the element. It increases the clickable area of a button. It is affected by the background color of the element. CSS: padding: 20px;
  3. Margin Margin is the external space outside the border of the element. It creates distance between one element and its neighbors. It is always transparent (you see the page background through it). CSS: margin: 20px;
  4. Block vs. Inline Elements This describes how elements sit on the page by default. Block Elements: These always start on a new line and take up the full width available (100%).
    5. Examples: <div>, <h1>, <p>, <tul>.
